The El Pollo Loco Guide

x

El Pollo Loco is a 2D platformer game. In this game, you control Pepe, a Mexican character. Wild chickens run towards you, which you must dodge, or else Pepe will take damage. You can find coins to collect or Tabasco bottles that you can throw at enemies. Hitting an enemy with a bottle causes them damage and eliminates them. A small tip: Save bottles for the boss, as you can only eliminate them using the bottles.

Game Controls

Arrow left: Move Left

Arrow right: Move right

Space: Jump

D: Throw bottle
